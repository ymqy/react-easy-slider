# react-easy-slider
Slider Component of React

## 设计
- 支持 Plugin 机制，组件可插拔设计
- 所有组件样式支持自定义
- 兼容 PC/H5，支持响应式
- 支持自定义过渡动画
- 支持多手势触控

## 性能优化
- 防止多余刷新
- 支持防抖、节流
- 严格按照 Tree-Shaking 模式开发
- 打包体积优化（Uglify）
